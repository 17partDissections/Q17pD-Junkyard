Tycoon

Project type: Computer game
First mention: 07.05.2024
Release date: 12.10.2024 (Version 0.4)
Last update: 06.11.2024 (Version 0.7)
Status: Cancelled
Github repo: 17partDissections/Tycoon

Platform: Web
Programming Language: C#
Engine: Unity 3D
Localization: English, Russian, Turkish


Iron Impact was one of my experimental projects and my second project for an unnamed web source.

This time I decided to parody those endless tycoon games.
I used a farm environment for the visuals. The player was an independent farm food salesman.

The main reason development took so long was the handmade assets. I made many of the models, textures, and sounds by myself. After 4-5 months of development, the game was pretty playable and good enough for a public audience. I optimized it really well — players could even run it on mobile web.
Buuut... the moderators of the web source didn't accept my project without giving any reason.
I suppose it's because I was under 18.



Installation is not available.
