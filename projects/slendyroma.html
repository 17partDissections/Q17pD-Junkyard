Programming Language: C#
Engine: Unity 3D
Localization: English

Slendyroma - is a frivolous parody of Slenderman, powered by Unity Engine, and it's my first game ever. I just wanted to make a funny game with my friend Roman Kahablet in there ;3
Main objective is to collect 8 green mushrooms (or documents) and avoid the mutated Roma.
Well, Roma is the main character too.... Yes, everyone in this game is Roma. I didn't really think about the plot.
Anyway, this game has 3 different maps with 3 different monsters.
Good luck collecting the mushrooms, but don't eat them! Some say that mushrooms are exactly what causes the mutation.
