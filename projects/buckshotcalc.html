BuckshotCalc

Project type: Desktop Program
First mention: 04.11.2024
Release date: 05.11.2024 (Version 0.1)
Last update: 01.12.2024 (Version 0.2)
Status: Active
Github repo: None

Platform: Windows
Programming Language: Python
Localization: English



BuckshotCalc is a helper tool for Buckshot Roulette. It keeps track of shell info so you don't have to remember everything.


Hot to use it:

1. Start of the round: At the beginning of a round, enter the number of live and blank shells like this:
5/2 (This means that 5 live shells and 2 blank shells).

2. During the round: Each time a shell fired, tell the program which shell was shot. It will then recalculate the chances for the next shell.

3. Shell knowledge: If you used a Magnifying Glass or a Burner Phone, you can enter what you learned. For example:
3b (This means that the 3rd shell is live).

4. Inverted shells: If a shell was inverted using the Inverter, just press the SHELL INVERTED button.
