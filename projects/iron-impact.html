Iron Impact

Project type: Computer game
First mention: 06.02.2024
Last update: 04.05.2024
Status: Cancelled
Github repo: None

Platform: Web
Programming Language: C#
Engine: Unity 3D
Localization: English, Russian, Turkish











Iron Impact was one of my experimental projects and my first project for an unnamed web source.
I hate the medieval fantasy genre in games, so I decided to make a medieval game myself :D

The gameplay was similar to World of Warcraft, but with a level system. The player had to walk through
a level and kill enemies. I implemented the player, enemy AI, weapon and consumable item systems,
but then I faced an optimization problem. I wanted to publish this game on a web source,
so it had to be very lightweight. However, it was very hard to fix. I also just lost interest in the project, so after three months of development, I cancelled it.



Installation is not available.
