ОПГ "Far'aCoast" - is a public not moderated Discord server created by Nor, OVERLORDGODS and Q17pD.
On this server you can play games with your teammates and blog your projects in your special channels(you should ask admin to create it for you)
here are some rules you should know before you chat:

1. be nice with others. being aggressive or toxic is not good.
2. insulting the server is forbidden.
3. don't flood or spam.
4. NSFW is forbidden.
5. if you're using too many commands, don't use the discussion channels etc. go to bot channels.
6. when you talking about something spoiler heavy, use the /spoiler tags. dont ruin the fun for others.
7. sometimes jokes get stretched too far. learn to know when enough is enough.
8. if your profile has NSFW content(name/pfp/banner), we advise you to change your profile.
there aren't many of them, so I think you can remember that.
please keep everything in order and have fun!
